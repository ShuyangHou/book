## 一、侯舒扬的工作内容

### 1. 项目初始化与框架搭建
- 创建项目仓库结构
- 配置 Git LFS 用于大文件管理(视频数据集)
- 编写项目文档:
  - `2026春季学期项目制实践实验指导书.md`
  - `LABELME_标注指南.md`

### 2. 数据预处理工具开发
开发了完整的数据处理 pipeline(`_tools/` 目录):

- **`quality_scan.py`**: 视频质量扫描,生成缩略图和质量报告
- **`reextract_bad_frames.py`**: 重新提取低质量帧
- **`build_ocr_dict.py`**: 构建OCR字典
- **`pick_top50.py`**: 从所有帧中选取top50用于标注
- **`auto_label_opencv.py`**: 基于OpenCV的自动标注工具
- **`count_labels.py`**: 统计标注数量
- **`build_dataset.py`**: 构建训练/验证数据集
- **`check_and_fix_labels.py`**: 检查并修复标注文件
- **`labelme2coco.py`**: 将LabelMe格式转换为COCO格式
- **`train_maskdino_r50.py`**: MaskDINO训练脚本
- **`predict_remaining.py`**: 批量预测剩余帧
- **`PIPELINE.md`**: 完整流程文档
- **`requirements_train.txt`**: 训练环境依赖

### 3. 标注与数据管理
- 负责视频046-067的预标注整理(共90个JSON文件)
- 维护 `annotations/` 目录下的所有CSV清单:
  - `video_manifest.csv`: 视频清单
  - `video_gt.csv`: 视频ground truth
  - `label_status.csv`: 标注状态跟踪
  - `low_quality_candidates.csv`: 低质量候选
  - `reextract_report.csv`: 重提取报告

### 4. 数据集构建
- 生成 `book_spine_dataset/coco/` 目录结构:
  - 训练集: 39张图片 + `instances_train.json`
  - 验证集: 10张图片 + `instances_val.json`
- 完成COCO格式转换,支持实例分割训练

### 5. 项目管理与协调
- 负责代码审查与合并(Merge PR #1)
- 协调整体进度
- 管理GitHub仓库

## 二、张梓晴的工作内容
